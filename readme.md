# 🧭 쉼따라 (Shimttara)

> 고령자·보행 약자를 위한 **쉼터 경유형 시니어 친화 보행자 네비게이션**

**쉼따라(Shimttara)**는 폭염·한파 등 기후 취약 상황에서  
고령자와 보행 약자가 보다 안전하게 이동할 수 있도록  
**스마트 쉼터 및 공공 편의시설을 경유하는 보행자 경로를 안내하는  
공공 서비스 지향 PoC(개념 검증) 프로젝트**입니다.

---

## 📌 프로젝트 추진 배경

- 폭염·한파 시 고령자 및 보행 약자의 이동 위험 증가
- 기존 내비게이션은 **‘최단 거리’ 중심**으로 설계되어  
  휴식·대피 공간을 고려하지 못하는 한계 존재
- 스마트 쉼터, 무더위·한파 쉼터 등 공공 인프라는 구축되어 있으나  
  👉 **실제 보행 경로와 연계된 서비스는 부족한 상황**

이에 따라 본 프로젝트는  
**“쉬어갈 수 있는 경로를 안내하는 보행자 네비게이션”**이라는 관점에서  
공공 데이터와 지도 서비스를 결합한 시범 모델을 제안합니다.

---

## 🎯 주요 기능

### 1. 쉼터 경유 보행자 경로 안내
- 일반 모드 / 기후 취약(한파·폭염) 모드 구분
- 기온 조건에 따라 **스마트 쉼터 또는 대피 쉼터를 경유하는 경로 제공**

### 2. 시니어 친화 UI·UX 설계
- 큰 글자, 큰 버튼, 단순한 화면 구성
- 터치 영역 확대 및 명확한 색 대비 적용
- 정보 과다를 지양하고 핵심 기능 중심 구성

### 3. 주변 공공 편의시설 안내
- ♿ 전동보장구(휠체어) 급속 충전소
- 🚻 공공 화장실
- 📍 현재 위치 기준 인근 시설 안내

### 4. 지도 레이어 선택 기능
- 스마트 쉼터
- 한파·무더위 쉼터
- 공공 화장실
- 전동보장구 충전소  
→ 이용 목적에 따라 필요한 정보만 선택적으로 표시

## 🖥️ 화면 구성

### 좌측 패널
- 출발지 / 도착지 입력
- 길찾기 / 다시입력
- 집 · 회사 · 현재 위치 단축 버튼
- 주변 편의시설 안내 버튼
- 지도 레이어 토글

### 우측 지도
- 네이버 지도 기반
- 쉼터 및 공공시설 마커 표시
- 보행자 경로 시각화
- 기온 정보 표시용 날씨 위젯(HUD)

---

## 🛠️ 적용 기술

### Frontend
- HTML / CSS / JavaScript (Vanilla)
- **Pretendard** 폰트 적용
- 글래스모피즘 기반 UI
- 시니어 접근성을 고려한 디자인 토큰 설계

### 지도·API 연동
- Naver Maps JavaScript API
- Tmap 보행자 경로 API
- OpenWeatherMap API
- 공공 데이터(JSON)
  - 스마트 쉼터
  - 무더위·한파 쉼터
  - 전동보장구 충전소
  - 공공 화장실
